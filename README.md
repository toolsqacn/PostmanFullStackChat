- Postman 编写 API 测试案例（RESTful API、SOAP）
- Postman 编写 API 流程（场景）测试案例
- Postman、Newman、和 Jenkins 实现持续集成 API 自动化测试
- 自动发布持续集成 HTML 测试报告
